# CLAUDE.md

## Project Overview

General-purpose full stack workspace using .NET Core (backend) and Angular (frontend).

## Tech Stack

- **Backend:** .NET Core / ASP.NET Core (C#)
- **Frontend:** Angular (TypeScript)
- **Platform:** Windows 11

## Conventions

### Backend (.NET Core)
- Follow standard C# naming conventions (PascalCase for public members, camelCase for locals)
- Use async/await for I/O-bound operations
- Prefer dependency injection via constructor injection
- Use `dotnet` CLI for build, run, and test commands

### Frontend (Angular)
- Follow Angular style guide (https://angular.dev/style-guide)
- Use Angular CLI (`ng`) for generating components, services, etc.
- Prefer standalone components where applicable
- Use TypeScript strict mode

### General
- Keep solutions simple and focused on the task at hand
- Prefer editing existing files over creating new ones
- Use Unix-style paths in shell commands (bash on Windows)
